\frametitle{What is PGFPlots ?}
A plot in a high-end \LaTeX \ document should:
\begin{itemize}
 \item be font-consistent,
 \item be high-quality,
 \item be based on (scientific) data,
 \item be scalable.
\end{itemize}
\pause
Hence, \texttt{PGFPlots} is what you need...
\end{frame}
